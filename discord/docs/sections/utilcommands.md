# 🔧 UtilCommands — 實用指令

常用的實用工具指令集，包含機器人資訊、伺服器資訊、延遲測試、更新日誌、教學指南等功能。

| 指令 | 說明 |
| --- | --- |
| `/help` | 查看所有可用指令 |
| `/info` | 查看機器人詳細資訊 |
| `/ping` | 測試機器人延遲 |
| `/serverinfo` | 查看當前伺服器資訊 |
| `/changelog` | 查看更新日誌 |
| `/git-commits` | 查看最近的 Git 提交紀錄 |
| `/tutorial` | 互動式新手教學 |
| `/nitro` | Nitro 禮物連結防護 |
