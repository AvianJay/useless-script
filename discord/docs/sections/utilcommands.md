# 🔧 UtilCommands — 實用指令

常用的實用工具指令集，包含機器人資訊、伺服器資訊、延遲測試、更新日誌、教學指南等功能。

| 指令 | 說明 |
| --- | --- |
| `/info help` | 查看所有可用指令 |
| `/info bot` | 查看機器人詳細資訊 |
| `/info ping` | 測試機器人延遲 |
| `/info user` | 查看指定用戶資訊 |
| `/info server` | 查看當前伺服器資訊 |
| `/info avatar` | 查看用戶頭像 |
| `/info banner` | 查看用戶橫幅 |
| `/info mention` | 取得指令提及格式 |
| `/info changelog` | 查看更新日誌 |
| `/info commits` | 查看最近的 Git 提交紀錄 |
| `/info tutorial` | 互動式新手教學 |
| `/nitro` | Nitro 禮物連結防護 |
