# 🚌 twbus — 台灣公車 / YouBike

即時查詢台灣公車到站資訊、路線總覽、YouBike 站點狀態，並支援收藏站牌。公車資料庫每小時自動更新一次。

| 指令 | 說明 |
| --- | --- |
| `/bus getroute` | 查詢指定路線的所有站牌與到站資訊 |
| `/bus getstop` | 查詢指定路線下特定站牌的即時到站資訊 |
| `/bus youbike` | 查詢指定 YouBike 站點的可借車數與空位 |
| `/bus favorites` | 查看並快速存取你收藏的站牌與 YouBike 站點 |

## 互動按鈕

查詢結果的嵌入訊息下方附有操作按鈕：

| 按鈕 | 說明 |
| --- | --- |
| `🔄` | 重新整理到站資訊（10 秒冷卻） |
| `❤️` | 加入 / 移除最愛（收藏上限可由伺服器設定） |
| `🗺️` | 在 Google 地圖上開啟站牌位置（若有座標） |

## 到站資訊說明

| 欄位 | 說明 |
| --- | --- |
| 預估到站 | 距離到站的分秒數，若顯示「進站中」表示車輛已進站 |
| 預計到站 | API 給出的預計時間（時:分格式） |
| 車輛狀態 | 列出接近中的車牌號碼及是否已滿 |
| 站序 | 此站牌在路線中的順序 |
| 座標 | 點擊可開啟 Google Maps |

## YouBike 資訊說明

| 欄位 | 說明 |
| --- | --- |
| 可借車數 | 目前可借走的腳踏車數量 |
| 可停空位 | 目前可停放的空格數 |
| 總停車格 | 站點的總容量 |
| 狀態 | 該站是否正常營運 |
| 最後更新 | YouBike 系統回報的資料更新時間 |

> **速率限制：**每位用戶每 10 秒只能查詢一次，重新整理按鈕同樣受此限制。
> **最愛上限：**預設每人最多 2 個最愛站牌與 2 個最愛 YouBike 站點，上限可由設定檔調整。
