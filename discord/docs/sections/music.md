# 🎵 Music — 音樂播放

透過 Lavalink 節點高品質播放音樂，支援隊列管理、循環播放（單曲/隊列）、隨機播放、搜尋自動完成、推薦歌曲、斷線自動離開、隊列儲存/恢復等功能，並支援多節點備援。

| 指令 | 說明 |
| --- | --- |
| `/music play` | 播放音樂（支援 URL 或關鍵字搜尋） |
| `/music search` | 搜尋音樂 |
| `/music skip` | 跳過當前曲目 |
| `/music shuffle` | 隨機播放隊列 |
| `/music loop` | 切換循環模式（單曲/隊列/關閉） |
| `/music recommend` | 獲取推薦歌曲 |
| `/music restore-queue` | 恢復之前的隊列 |
| `/music nodes` | 查看可用的 Lavalink 節點 |
