# 🤖 AutoModerate — 自動管理 管理員自動化

全自動化的伺服器保護系統，包含反垃圾訊息 (Anti-Spam)、反突襲 (Anti-Raid)、反使用者安裝垃圾訊息、邀請連結偵測、詐騙陷阱、過多表情偵測、逃避懲處偵測等多種自動化管理功能。

| 指令 | 說明 |
| --- | --- |
| `/automod quick-setup` | 快速設定自動管理規則 |

> **功能清單：**Anti-Spam、Anti-Raid、Anti-UISpam、Anti-Invite Link、ScamTrap、過多表情偵測、逃避懲處偵測、AutoMod 事件整合。Anti-Invite Link 支援「允許本伺服器連結」設定，可只阻擋外部伺服器邀請。可透過 [網頁面板](/panel/) 進行詳細配置。
