# 💰 Economy — 經濟系統

完整的雙幣制虛擬經濟系統。全域幣可跨伺服器使用，伺服幣僅限當前伺服器。包含每日/每小時獎勵、商店買賣、轉帳、匯率機制、通膨/通縮模型、支票系統等。

| 指令 | 說明 |
| --- | --- |
| `/economy daily` | 領取每日獎勵 |
| `/economy hourly` | 領取每小時獎勵 |
| `/economy balance` | 查看餘額 |
| `/economy buy` | 購買商店物品 |
| `/economy sell` | 出售物品 |
| `/economy trade` | 與其他使用者交易/轉帳 |
| `/economy history` | 查看交易紀錄 |
| `/economymod` | 管理員經濟管理工具 |

> **防濫用機制：**系統會追蹤管理員注入的貨幣並設有上限控制，確保經濟平衡。
