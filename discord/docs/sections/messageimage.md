# 📸 MessageImage — 訊息截圖

將 Discord 訊息轉換為一些奇怪的圖片(?)。支援 Upvoteboard 功能：當訊息獲得足夠讚數後，自動截圖並發送至指定頻道保存精華。

- 指令?`y!ss (回覆訊息)` | 說明?截圖指定訊息
- 指令?`訊息截圖器(訊息應用程式選單)` | 說明?截圖指定訊息
- 指令?`糟糕的Make it a quote(訊息應用程式選單)` | 說明?仿Make it a quote圖片，但是比較糟糕。
- 指令?`這傢伙在說什麼呢(訊息應用程式選單)` | 說明?截圖並在下方隨機加入一張對話框圖片。
- 指令?`/upvoteboard` | 說明?設定 Upvoteboard 頻道
