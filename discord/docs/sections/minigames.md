# 🃏 MiniGames — 迷你遊戲 娛樂

集合多種卡牌、賭博與即時遊戲，整合伺服器幣及全域幣進行下注與派彩。

在 Discord Explore 大廳可透過右側的賭場接待員進入全域賭場。Explore 內的九款賭場遊戲使用全域幣；彩票與 `/games lottery` 的全域模式共用同一獎池與開獎回合。

| 指令 | 說明 |
| --- | --- |
| `/games big2` | 大老二 — 經典撲克牌遊戲 |
| `/games tower` | 爬塔 — 賭博挑戰遊戲 |
| `/games slots` | 拉霸機 — 對子與三連線派彩 |
| `/games highlow` | 比大小 — 猜下一張牌較大或較小 |
| `/games blackjack` | 21 點 — 對抗莊家，可要牌、停牌或加倍 |
| `/games roulette` | 俄羅斯輪盤 — 投注紅黑、單雙、大小或單一號碼 |
| `/games dice` | 骰子 — 下注後用按鈕選擇 1～6 的點數 |
| `/games coinflip` | 擲硬幣 — 下注後用按鈕選擇正面或反面 |
| `/games scratchcard` | 刮刮樂 — 刮開 3x3 票面取得隨機獎項 |
| `/games lottery` | 累積彩票 — 購買 00～99 號碼；每輪在下一個整點開獎 |
| `/games doom` | 在 Discord 內遊玩 DOOM |
