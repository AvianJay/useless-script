uooxwu Python Client

uooxwu 是這個 API 的 Python client，包好了 REST 呼叫、Socket.IO 事件監聽，以及常用的 dataclass model。

適合 Python bot、CLI 工具、後端服務直接接 OXWU Proxy API。你不用自己手刻 request header、JSON parsing、或 Socket.IO 事件轉型。

Quick Start

import uooxwu

client = uooxwu.Client(
    url="https://oxwu.avianjay.sbs",
    api_key="YOUR_API_KEY",
)

warning = client.get_warning()
print(warning.time)
print(warning.max_intensity)
print(warning.arrival_times)

town_map = client.get_town_map()
print(town_map["6500600"].name)

@client.event()
def proxy_warning_update(event: uooxwu.WarningUpdateEvent):
    print(event.time)
    print(event.arrival_times)
    print(event.estimated_intensities)

client.connect(wait=True)

Client

項目 型別 說明
Client(url, api_key=None, timeout=10.0, socket_transports=None) class 建立 client。預設 Socket.IO transport 為 ["websocket", "polling"]。
get_warning() Warning 取得目前 warning 快取並轉成 dataclass model。
get_report() Report 取得目前 report 快取並轉成 dataclass model。
get_screenshot(type_) bytes 取得 report 或 warning PNG。
load_builtin_town_map() dict[str, Town] 讀取套件內建的 town map，不需要打 API。
fetch_town_map() dict[str, Town] 向 /api/town-map 取得遠端 town map。
get_town_map(source="builtin"|"remote"|"auto") dict[str, Town] 整合版介面。auto 會先抓遠端，失敗時 fallback 到內建資料。
event() / on(name) decorator 註冊 Socket.IO 事件監聽器。
connect(wait=False) socketio.Client 建立 Socket.IO 連線。wait=True 會阻塞直到斷線。
wait() method 等待既有的 Socket.IO 連線。
disconnect() method 中斷 Socket.IO。
close() method 關閉 Socket.IO 與底層 HTTP session。

Events

事件名稱 handler 參數 說明
connect 無參數 Socket.IO 連線成功。
disconnect 無參數 Socket.IO 斷線。
proxy_warning_update WarningUpdateEvent 新的 warning 開始。
proxy_warning_updated WarningUpdatedEvent 同一輪 warning 的後續更新。
proxy_report_update ReportUpdateEvent 新的 report 推送。

事件 decorator 範例

@client.event()
def connect():
    print("connected")

@client.event()
def proxy_warning_update(event: uooxwu.WarningUpdateEvent):
    print(event.time)
    print(event.warning.max_intensity if event.warning else None)

@client.on("proxy_warning_updated")
def handle_warning_updated(event: uooxwu.WarningUpdatedEvent):
    print(event.arrival_times)

Models

套件會把常用 payload 轉成 dataclass 物件。你通常只要認這幾個：

Warning WarningLocation WarningRecord WarningUpdateEvent WarningUpdatedEvent
Report ReportData ReportIntensityArea ReportStationGroup Town
Model 常用屬性 說明
Warning time, depth, magnitude, max_intensity, arrival_times, estimated_intensities, raw 完整 warning payload。
WarningUpdateEvent time, parts, warning, arrival_times 對應 proxy_warning_update。
WarningUpdatedEvent time, warning, arrival_times 對應 proxy_warning_updated。
Report ok, report, raw 完整 report payload。
Town town_id, name, latitude, longitude town map 單筆資料。

Town Map

uooxwu 內建一份 town_id.json，所以你不一定要每次都打 API 才能把 town_id 轉回地名。

town_map = client.get_town_map()
print(town_map["6500600"].name)

remote_map = client.get_town_map(source="remote")
auto_map = client.get_town_map(source="auto")

Example

import uooxwu

client = uooxwu.Client(
    url="{{ request.host_url.rstrip('/') }}",
    api_key="YOUR_API_KEY",
)

@client.event()
def connect():
    print("connected")

@client.event()
def proxy_warning_update(event: uooxwu.WarningUpdateEvent):
    print("warning:", event.time)
    print("arrival:", event.arrival_times)
    print("estimated:", event.estimated_intensities)

@client.event()
def proxy_report_update(event: uooxwu.ReportUpdateEvent):
    print("report:", event.time)

if __name__ == "__main__":
    warning = client.get_warning()
    print(warning.time)
    print(warning.raw)
    client.connect(wait=True)
