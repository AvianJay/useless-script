OXWU API Docs

這份文件涵蓋資料查詢、截圖下載、登入後帳號資訊，以及 WebSocket 代理事件。登入後可以在 dashboard 直接取得與重設 API Key。

OpenAPI JSON: {{ openapi_url }}

Authentication

一般 API 需在 request header 帶入 X-API-Key。帳號管理 API 使用 Discord OAuth2 登入後的 session cookie。

點數規則

  • 資料 API: {{ "%.2f"|format(costs.api) }} 點 / 次
  • 截圖 API: {{ "%.2f"|format(costs.screenshot) }} 點 / 次
  • WebSocket: {{ "%.4f"|format(costs.ws_per_sec) }} 點 / 秒，5 秒結算一次

快速測試

curl -H "X-API-Key: YOUR_API_KEY" \
{{ request.host_url.rstrip('/') }}/api/data/report

REST Endpoints

GET/api/data/<report|warning>

回傳目前快取的 JSON 資料。

GET/api/screenshot/<report|warning>

回傳 PNG 截圖內容。

GET/api/me

登入後取得目前帳號資訊、點數與 API Key。

POST/api/me/reset-key

登入後重設 API Key，舊 key 立即失效。

GET/api/docs

回傳目前這份說明文件（HTML）。

GET/openapi.json

回傳 OpenAPI 3.1 規格（JSON）。

Upstream Mapping (from main.js)

Proxy 會向上游 Electron HTTP 服務（預設 http://127.0.0.1:10281）抓資料並更新快取，對應如下：

  • report 快取來源: /getReportInfo
  • warning 快取來源: /getWarningInfo
  • 截圖流程: 先呼叫 /gotoReport 或 /gotoWarning，再呼叫 /screenshot

上游事件 reportTimeChanged / warningTimeChanged 觸發後，proxy 會更新快取並轉發給客戶端：

  • proxy_report_update
  • proxy_warning_update

WebSocket

Socket.IO 連線時，帶入 X-API-Key header 或 query string 的 api_key。伺服器會轉發以下事件：

  • proxy_report_update
  • proxy_warning_update
const socket = io("{{ request.host_url.rstrip('/') }}", {
  extraHeaders: { "X-API-Key": "YOUR_API_KEY" }
});
