# OXWU API
地牛 Wake Up! 的 API
# 開始使用
## 前提
* 你必須已經安裝了地牛 Wake Up!
## 安裝
Windows
* 直接開啟patch.bat即可修補。

Linux/MacOS
* 將`main.js`把地牛Wake Up! 本身的`main.js`取代，
# 已測試的版本
* v4.1.0
## Discord Webhook
在 地牛Wake Up! 的設定內，在其他類別那裡有個連動設定，選擇此資料夾的discord.py。如果無法執行請先`chmod +x discord.py`

**一定要把僅執行一次打開！！！**

Windows 用戶請先使用pyinstaller打包好才可使用。`build.bat`
